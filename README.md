# Data Wrangling Final Project

This repository contains the deliverables for my Module 8 Data Wrangling final project. The project combines primary education enrollment and GDP per capita data for Peru from 2001 through 2020.

## Files

- `peru_primary_enrollment_gdp_2001_2020.csv`: Final clean CSV dataset.
- `peru_education_gdp.db`: SQLite database containing the final dataset.
- `peru_primary_enrollment_gdp_description.json`: JSON data dictionary describing the CSV columns.
- `Module8_Data_Wrangling_Final_Project_Peru.Rmd`: Commented R Markdown notebook used to perform the wrangling steps.

## Country Selected

Peru

## Years Covered

2001-2020

## Main Question

Did primary education enrollment increase as GDP per capita improved over the past 20 years?
